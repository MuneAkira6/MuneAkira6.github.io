一、操作符

1. void

作为函数返回值类型，表示不关注返回值类型，可以是任意值

2. 非空断言 !

3. 键值获取 keyof

使用场景：任意定义一个 key 变量去对象里取 value 时，通常会不允许取。故需要手动指定。
那如果无法获取显式目标对象的类型时？keyof typeof

二、泛型工具

Partial

将泛型中全部属性变为可选的。

Required

将泛型中全部属性变为必选的。

Record<K, T>

常用于定义对象。 Record<string, unknown>

Pick<T, K>

取键值对。
Pick<Animal, "name" | "age">

Omit<T, K>

去键值对。
Omit<Animal, 'name'|'age'>
